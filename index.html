HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • use the ul to make unlist and li to create listed items

CSS

  • Margin: indicates how much space outside an element
  • padding: add space around the content inside the element

Git

  • git status: check what branch currently on
  • git checkout switch to a branch, use -b if a new branch needs to be created
  • use cd .. to go back directorys
  • git add -A: to commit changes
  • git commit -m: commit to respository
  • touch xxx.xxx to create new file

JavaScript

  • Use "const" to declare a variable
  • Use a "for" to make predictable patterns
  • The first statement for the "for" loop is the starting point
  • The second statement for the "for" loop is the condition
  • The final statement is what allows the array to iterate over each item
  • x++ is a shortcut way to write x+1
  • console.log shows the code
  • use /* and */ to comment out a block of code
